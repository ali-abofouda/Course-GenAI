# GenAI Course Study Notes

## ملخص سريع
- عدد الأقسام: 52
- عدد المحاضرات: 224
- المدة الإجمالية: 57 ساعة و37 دقيقة

## ماذا ستتعلم في الكورس؟
1. أساسيات Python
2. NLP الكلاسيكي (Tokenization, Stemming, Lemmatization, TF-IDF, Word2Vec)
3. Deep Learning for NLP (ANN, RNN, LSTM, GRU, Seq2Seq, Attention, Transformers)
4. Generative AI وLLMs
5. LangChain وLangGraph وMCP
6. بناء تطبيقات عملية بـ Streamlit وRAG وAgents
7. النشر على منصات مختلفة (Streamlit Cloud, Hugging Face, AWS)

## مسار التعلم المقترح
